# AI-Medical-Assistant

一个基于大模型的智能医疗助手项目，支持医疗问答、知识检索（RAG）与预约管理。

## 仓库说明

当前仓库代码位于子目录：

- `medical-ai-assistant-github/`

请进入该目录查看完整项目与文档。

## 快速开始

```powershell
cd medical-ai-assistant-github
```

查看完整说明：

- [项目完整 README](./medical-ai-assistant-github/README.md)

## 项目功能

- 多 Agent 医疗对话流程
- 医疗知识 RAG 检索（向量 + 混合检索）
- 会话记忆与安全词审查
- 预约挂号流程（创建 / 查询 / 取消）
- FastAPI 后端 + Vue3 前端
